## 1. 커서 이동

- `Ctrl + a` : 커서를 줄의 시작부분으로 이동한다.
- `Ctrl + e` : 커서를 줄의 끝부분으로 이동한다.
- `Ctrl + f` : 커서를 한 글자 앞으로 이동한다.
- `Ctrl + b` : 커서를 한 글자 뒤로 이동한다.
- `Alt + f` : 커서를 한 단어 앞으로 이동한다.
- `Alt + b` : 커서를 한 단어 뒤로 이동한다.
- `Ctrl + l` : 화면을 지우고 커서를 왼쪽 상단 모서리로 이동한다. `clear` 명령과 동일하다.

---

## 2. 텍스트 수정하기

- `Ctrl + d` : 커서 위치에서 문자를 삭제한다.
- `Ctrl + t` : 커서 위치의 문자를 앞의 문자와 바꾸기(교환)한다.
- `Alt + t` : 커서 위치의 단어를 앞의 단어와 바꾸기(교환)한다.
- `Alt + l` : 커서 위치에서 단어 끝까지의 문자를 소문자로 변환한다.
- `Alt + u` : 커서 위치에서 단어 끝까지의 문자를 대문자로 변환한다.

---

## 3. 텍스트 잘라내기 및 붙여넣기

- `Ctrl + k` : 커서 위치에서 줄 끝까지 텍스트를 삭제한다.
- `Ctrl + u` : 커서 위치에서 줄의 시작까지 텍스트를 삭제한다.
- `Alt + d` : 커서 위치에서 현재 단어 끝까지의 텍스트를 삭제한다.
- `Alt + Backspace` : 커서 위치에서 현재 단어의 시작 부분까지 텍스트를 삭제한다.
커서가 단어의 시작 부분에 있으면 이전 단어를 삭제한다.
- `Ctrl + y` : 삭제한 텍스트를 붙여넣는다.
